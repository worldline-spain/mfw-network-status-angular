Dependencies

  • ionic
  • ngCordova

Description

This module provides an abstraction of ngCordova's $cordovaInAppBrowser plugin.

Plugins

This module requires the following Cordova plugins:

  • cordova-plugin-inappbrowser
  • cordova-plugin-safariviewcontroller

Features

iOS

  • Uses SFSafariViewController when available, UIWebViewController otherwise.
  • iOS 9+ provided WKWebView with huge performance improvements over UIWebView.
