$mwfiBrowser
service in module mfw-ionic.inapp-browser

Description

InApp browser service that allows you to open external URLs using native browser or embedded webviews.

Methods

  • init()

    Initializer method, to be called when module is loaded.

    It detects whether SafariViewController is available or not.

  • open(url, target, options)

    Open a new browser window with given URL, target and options.

    Internally, this method calls InAppBrowser or SafariViewController depending on its availability (iOS 9+).

    Parameters
    ParamTypeDetails
    urlstring

    URL to open

    target
    (optional)
    		string

    The target in which to load the URL, an optional parameter that defaults to configured options.target if set or _self.

    • _self: Opens in the Cordova WebView if the URL is in the white list, otherwise it opens in the InAppBrowser.
    • _blank: Opens in the InAppBrowser.
    • _system: Opens in the system's web browser app.
    options
    (optional)
    		object

    InAppBrowser options for the InAppBrowser and/or SafariViewController options.

    Returns
    Promise<InAppBrowser>

    Promise that will resolve when browser is opened. It will resolve with new InAppBrowser instance if using target _blank.
